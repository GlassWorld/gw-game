# 스킬 정리 문서

## 목적

이 문서는 현재 프로토타입에 들어간 캐릭터별 스킬 풀을 빠르게 확인하기 위한 문서다.

## 공통 규칙

- 각 캐릭터는 5개 스킬을 가진다.
- 대기소에서 3개를 선택한다.
- 선택 순서대로 `Q / W / E`에 배치된다.
- 현재는 아이콘, 설명, 데미지, 쿨타임, MP 소모, 예시 미리보기가 스킬 선택 화면에 표시된다.

## 검사 스킬

- `블레이드 웨이브`
  - 전방에 검기를 부채꼴로 날림
- `와류 베기`
  - 주변 근접 범위 회전 베기
- `가드 브레이크`
  - 전방 좁은 범위를 강하게 내려침
- `크로스 슬래시`
  - 전방 교차 검격 발사
- `대지 가르기`
  - 지정 지점에 충격 장판 생성

## 마법사 스킬

- `아케인 팬`
  - 전방 다중 마력탄
- `서리 고리`
  - 주변 냉기 폭발
- `메테오 낙하`
  - 지정 지점 강력 낙하 마법
- `마나 랜스`
  - 강한 관통 마탄
- `오브 폭산`
  - 짧은 지연 후 폭발하는 구체

## 성기사 스킬

- `홀리 볼트`
  - 성속성 투사체 발사
- `축성`
  - 주변 성역 충격
- `방패 강타`
  - 전방 근접 강타
- `심판의 창`
  - 집중된 성창 발사
- `성역 폭발`
  - 지정 지점 성광 폭발

## 현재 상태 메모

- 현재는 이펙트보다 발동 구조와 범위 확인이 우선이다.
- 스킬 방향은 보스 자동 추적이 아니라 마우스 방향/위치 기준이다.
- 향후에는 스킬 강화와 캐릭터별 빌드 시너지가 별도 규칙으로 추가될 수 있다.
