# 붉은 심판자 문서

## 1. 기본 정보

- 보스 ID: `red-adjudicator`
- 보스명: `붉은 심판자`
- 지역명: `잔열의 재판장`
- 타이틀: `불완전한 심문 집행자`
- 주사위 면: `I`
- 대표 색상: `#ff7f6a`

## 2. 한 줄 컨셉

- 무너진 재판장의 불씨를 끌어안은 채, 탄막과 장판, 돌진으로 플레이어를 압박하는 추적형 심판 보스

## 3. 전투 역할

- 플레이어에게 요구하는 핵심 대응:
  - 장판과 돌진의 텔레그래프를 읽고, 짧은 빈틈에 딜을 넣는 대응
- 강하게 드러나야 하는 보스 성격:
  - 위압적이고 집요하며, 재판을 강행하는 집행자
- 이 보스가 다른 보스와 구분되는 포인트:
  - 정면 탄막 압박과 일직선 돌진의 교차 압박

## 4. 지역 / 지형

- 지역 분위기:
  - 붉은 잔열과 잿빛 파편이 남은 붕괴 재판장
- 주요 지형 요소:
  - 중앙 심판대, 균열 바닥, 불씨 흔적
- 이동에 영향을 주는 요소:
  - 현재 프로토타입에서는 없음
- 기믹 오브젝트:
  - 현재 프로토타입에서는 없음

## 5. 컷씬 정의

### 진입 컷씬

- 컷 1:
  - `잔열의 재판장`
  - 붕괴한 법정의 잔향과 남은 열기를 보여준다.
- 컷 2:
  - `붉은 심판자`
  - "심판은 끝나지 않았다. 불씨가 꺼지기 전까지 너를 재단하겠다."

### 사망 컷씬

- 컷 1:
  - 불씨가 무너진 심판대 아래로 흩어진다.
- 컷 2:
  - 재판장의 열기가 식으며 대기소 복귀 연출로 이어진다.

## 6. 페이즈 구조

### 페이즈 1

- 시작 조건:
  - 전투 시작 시
- 종료 조건:
  - HP가 일정 이하로 감소하면 페이즈 2 진입
- 주요 패턴:
  - 탄막 사격
  - 장판 폭발
  - 직선 돌진
- 플레이 감각:
  - 기본 패턴 학습과 회피 리듬 적응 단계

### 페이즈 2

- 시작 조건:
  - 보스 HP 임계치 도달
- 종료 조건:
  - 보스 처치
- 주요 패턴:
  - 같은 패턴을 더 빠르거나 빡빡하게 조합
- 플레이 감각:
  - 텔레그래프를 더 빨리 읽어야 하는 압박 단계

## 7. 패턴 정의

### 패턴 A

- 이름: `탄막 사격`
- 목적:
  - 플레이어의 이동 경로를 강제로 꺾음
- 텔레그래프:
  - 보스 발밑 패턴 마커, 패턴 시작 문구
- 실제 판정:
  - 전방 또는 확산 방향 탄막 발사
- 실패 시 위험:
  - 연속 피격으로 체력 손실

### 패턴 B

- 이름: `장판 폭발`
- 목적:
  - 특정 위치에 오래 머무르지 못하게 함
- 텔레그래프:
  - 강화된 경고색 장판
- 실제 판정:
  - 일정 시간 후 원형 범위 폭발
- 실패 시 위험:
  - 강한 순간 피해

### 패턴 C

- 이름: `심판 돌진`
- 목적:
  - 직선 압박으로 회피 타이밍 강요
- 텔레그래프:
  - 돌진 전 방향선 표시
- 실제 판정:
  - 보스가 목표 방향으로 빠르게 돌진
- 실패 시 위험:
  - 큰 피해와 위치 붕괴
