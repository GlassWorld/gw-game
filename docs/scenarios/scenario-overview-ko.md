# 시나리오 정리 문서

## 프로젝트 톤

`RBB`는 긴 서사보다는, 보스전 루프를 자연스럽게 묶어주는 짧은 설정과 컷씬 문구 중심의 구조를 우선한다.

## 기본 시나리오 흐름

1. 플레이어는 `RBB` 세계에서 다음 보스를 상대하기 위해 진입한다.
2. 주사위가 굴러가며 이번 전투의 보스와 지역이 결정된다.
3. 대기소에서 세팅을 마치고, 연습 공간에서 감각을 확인한다.
4. 보스 지역에 진입하며 짧은 진입 컷씬을 본다.
5. 보스를 쓰러뜨리면 사망 컷씬 후 대기소로 복귀한다.
6. 남은 보스가 있다면 다음 보스를 다시 선택한다.
7. 모든 보스를 쓰러뜨리면 이번 런이 종료된다.

## 현재 문서화 원칙

- 세계관 전체를 길게 쓰지 않는다.
- 화면 문구와 컷씬에 직접 쓰일 문장 위주로 정리한다.
- 보스별 시나리오는 각 보스 문서에서 관리한다.

## 현재 구현 상태

- 시작 화면: `RBB` + `시작하기`
- 보스 진입 컷씬: 각 보스별 2컷 구조
- 보스 사망 컷씬: 각 보스별 2컷 구조
- 런 클리어 시 홈 화면에 요약 배너 표시
